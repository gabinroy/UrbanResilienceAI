### Prerequisites
- Node.js & npm
- Firebase CLI (`npm install -g firebase-tools`)

```bash
# Clone repo
git clone https://github.com/gabinroy/UrbanResilienceAI.git
cd UrbanResilienceAI

# Install dependencies
````
